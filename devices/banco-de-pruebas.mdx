The Banco de Pruebas is a static test stand for measuring rocket motor performance during ground testing.

## Overview

The test bench provides real-time monitoring of:

- **Thrust (Fuerza)** - Motor thrust in Newtons
- **Temperature** - Chamber/nozzle temperature
- **Pressure** - Combustion chamber pressure
- **Remote Ignition** - Wireless ignition trigger

## Data Streams

### Telemetry Channel: `banco-datos`

**WebSocket Channel:** `banco-datos`  
**Update Rate:** 500ms  
**Database Table:** `prueba_estatica_0`  
**Endpoint:** `GET/POST /api/bancodepruebas`

<Accordion title="Data Fields">

| Field | Type | Description | Units |
|-------|------|-------------|-------|
| `id` | integer | Sequential record ID | - |
| `date` | string | Timestamp date | - |
| `time` | string | Timestamp time | - |
| `fuerza` | float | Measured thrust force | Ns (Newton-seconds) |
| `temperatura` | float | Temperature reading | °C |
| `presion` | float | Chamber pressure | Psi |
| `id_prueba` | integer | Test sequence identifier | - |

</Accordion>

### Ignition Command Channel

**WebSocket Channel:** `ignicion-estado`  
**Endpoint:** `POST /api/ignicion` (send), `GET /api/ignicion` (check)

<Accordion title="Ignition Protocol">

**Command Format:**
```json
{
  "command": "IGNICION"
}
```

**State Broadcast:**
When ignition is triggered, the backend:
1. Broadcasts `{ command: "IGNICION" }` to all clients
2. Waits 1000ms
3. Broadcasts `{ command: null }` to reset state

**Implementation:** `src/websocket.js:48-56`

</Accordion>

## Hardware Communication

The test bench uses Arduino connected via USB serial.

### Python Data Bridge

**Script:** `src/arduino/bancodepruebas.py`  
**Serial Port:** `/dev/ttyACM0`  
**Baud Rate:** 9600

<Accordion title="Serial Data Format">

Arduino sends comma-separated values:

```
fuerza,temperatura,valor2,valor3,presion
```

The Python script parses indices:
- `valores[0]` → `fuerza`
- `valores[1]` → `temperatura`
- `valores[4]` → `presion`

**Payload Construction:**
```python
data = {
    "id_prueba": 0,
    "fuerza": float(valores[0]),
    "temperatura": float(valores[1]),
    "presion": float(valores[4])
}
```

**Location:** `src/arduino/bancodepruebas.py:58-65`

</Accordion>

## Data Flow

1. Arduino reads load cell, thermocouple, and pressure sensor
2. Arduino sends `fuerza,temperatura,_,_,presion` via serial
3. Python script parses and posts to `POST /api/bancodepruebas`
4. Backend stores in `prueba_estatica_0` table
5. WebSocket broadcasts on `banco-datos` channel every 500ms
6. Frontend dashboard updates charts in real-time

### Ignition Flow

1. User clicks "Ignición" button in dashboard
2. Frontend sends `POST /api/ignicion` with `{command: "IGNICION"}`
3. Backend sets `ignicionState = true` and broadcasts to all clients
4. Python script polling loop receives command
5. Python script sends `"IGNICION"` string to Arduino via serial
6. Arduino triggers igniter circuit
7. Backend auto-resets ignition state after 1 second
