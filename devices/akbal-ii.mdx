AKBAL-II is a rocket system with real-time telemetry monitoring, 3D rocket orientation visualization, and mission phase tracking.

## Overview

The AKBAL-II dashboard (`frontend/src/pages/Dashboard.jsx`) provides a SpaceX-inspired interface for monitoring rocket flights:

- Live video/image feeds
- 3D rocket model with orientation visualization
- Real-time telemetry charts
- GPS tracking with live map updates
- Barometric altitude gauge
- Battery status monitoring

## Data Streams

### Primary Telemetry Channel: `nuevos-datos`

**WebSocket Channel:** `nuevos-datos`  
**Update Rate:** 100ms  
**Database Table:** `data`  
**Endpoint:** `GET/POST /api/data`

### Battery Monitoring Channel: `baterias`

**WebSocket Channel:** `baterias`  
**Update Rate:** 5 seconds  
**Database Table:** `battery_status`  
**Endpoint:** `GET/POST /api/batteries`

## Mission States

The `mission_state` field tracks the rocket's flight phase:

| State | Index | Description |
|-------|-------|-------------|
| **Preflight** | 0 | Pre-launch initialization |
| **Lift-off** | 1 | Motor ignition and launch |
| **Apogee** | 2 | Maximum altitude reached |
| **Drogue** | 3 | Drogue parachute deployed |
| **Main** | 4 | Main parachute deployed |
| **Land** | 5 | Touchdown complete |

## Data Subscription

The dashboard uses WebSocket for real-time updates:

```javascript
// frontend/src/pages/Dashboard.jsx:155
const rawData = useWebsocket('nuevos-datos', SOCKET_URL);
```

Data is processed and mapped to state arrays in `useEffect` hook (lines 157-188).
