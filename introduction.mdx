## Welcome to SAFI ControlHub

SAFI ControlHub is a comprehensive platform for monitoring and controlling rocket systems in real-time. Built for aerospace teams, it provides live telemetry visualization, test bench control, and multi-device management through an intuitive web interface.

## Supported Devices

ControlHub supports multiple rocket and test systems:

- **XITZIN-II** - Primary rocket telemetry system with battery monitoring
- **AKBAL-II** - Secondary rocket platform with enhanced sensors
- **Laniakea** - Advanced telemetry and data collection system
- **Banco de Pruebas** - Static test bench for engine performance testing
- **Filamentadora** - Composite filament winding system

<Note>
Each device streams real-time data through dedicated WebSocket channels with configurable update intervals.
</Note>

## Platform Architecture

The platform consists of three main components:

1. **Backend API** - Express.js server with PostgreSQL database and WebSocket support
2. **Frontend Dashboard** - React-based UI with real-time data visualization using D3.js and Three.js
3. **Hardware Integration** - Serial port communication for direct device connectivity

<Info>
ControlHub is designed for both local network deployment and cloud-based operations, with support for multiple concurrent users and device streams.
</Info>
