## Overview

SAFI ControlHub provides telemetry data endpoints for multiple aerospace systems:

- **AKBAL-II** - Rocket telemetry and battery status
- **XITZIN-II** - Rocket telemetry and battery status
- **Banco de Pruebas** - Test bench data
- **Laniakea** - Laniakea system telemetry

All telemetry endpoints accept and return data in JSON format.

---

## AKBAL-II Telemetry

### Get AKBAL-II Data

Retrieve the latest telemetry data from AKBAL-II (limited to 150 most recent records).

```http
GET /api/data
```

#### Response

Returns an array of telemetry records.

<ResponseField name="id" type="integer">
  Record identifier
</ResponseField>

<ResponseField name="date" type="string">
  Date in YYYY-MM-DD format
</ResponseField>

<ResponseField name="time" type="string">
  Time in HH:MM:SS format
</ResponseField>

<ResponseField name="altitude" type="number">
  Altitude measurement
</ResponseField>

<ResponseField name="temperature" type="number">
  Temperature measurement
</ResponseField>

<ResponseField name="pressure" type="number">
  Pressure measurement
</ResponseField>

<ResponseField name="velocity" type="number">
  Velocity measurement
</ResponseField>

<ResponseField name="latitude" type="number">
  GPS latitude coordinate
</ResponseField>

<ResponseField name="longitude" type="number">
  GPS longitude coordinate
</ResponseField>

<ResponseField name="accel_x" type="number">
  Acceleration on X-axis
</ResponseField>

<ResponseField name="accel_y" type="number">
  Acceleration on Y-axis
</ResponseField>

<ResponseField name="accel_z" type="number">
  Acceleration on Z-axis
</ResponseField>

<ResponseField name="mission_state" type="number">
  Current mission state identifier
</ResponseField>

<ResponseField name="air_brake_angle" type="number">
  Air brake angle position
</ResponseField>

## Error Handling

All data endpoints return consistent error responses:

### 400 Bad Request

Returned when required parameters are missing:

```json
{
  "message": "Faltan parámetros en la solicitud."
}
```

Or for Laniakea:

```json
{
  "message": "Faltan parámetros obligatorios en la solicitud."
}
```

### 500 Internal Server Error

Returned when database operations fail:

```json
{
  "message": "Error al obtener los datos."
}
```

Or:

```json
{
  "message": "Error al insertar los datos."
}
```
